# Features:
* Barbers.
* Saveable Outfits.
* Camera Rotating.
* Clothing Stores.
* 3 Different Camera Angles.
* Configurable Ped Selection.

* /hat, /glasses, /mask (See the commands section below.)
# Commands
* /hat - Toggles the hat On/Off.
* /mask - Toggles the hmaskat On/Off.
* /glasses - Toggles the glasses On/Fff
